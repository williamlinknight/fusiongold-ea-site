セットアップガイド Setup Guide

FusionGold M5 のインストールから運用開始までの手順を解説します。 Step-by-step instructions from installation to launching FusionGold M5.

所要時間: 約15〜30分 Time required: Approximately 15–30 minutes

01

Step 1: EAファイルをダウンロードStep 1: Download the EA File

物々交換が成立後、EAファイル（.ex5）と設定ガイド（PDF）をお送りします。ファイルはメールにてお届けします。After the barter agreement is finalized, we will send you the EA file (.ex5) and setup guide (PDF) via email.

💡 ヒント 💡 Tip メールが届かない場合は迷惑メールフォルダをご確認ください。If you don't see the email, please check your spam folder.
02

Step 2: MT5にEAをインストールStep 2: Install the EA on MT5

MT5プラットフォームを開き、メニューから「ファイル」→「データフォルダを開く」を選択します。表示されたフォルダ内の「MQL5」→「Experts」フォルダに、受け取った.ex5ファイルをコピーします。Open MT5, go to "File" → "Open Data Folder". Navigate to "MQL5" → "Experts" and copy the .ex5 file there.

💡 ヒント 💡 Tip MT5を再起動すると、ナビゲーターウィンドウの「エキスパートアドバイザー」にFusionGold M5が表示されます。After restarting MT5, FusionGold M5 will appear under "Expert Advisors" in the Navigator window.
03

Step 3: EAをチャートに適用Step 3: Apply the EA to a Chart

XAUUSDのM5チャートを開き、ナビゲーターからFusionGold M5をチャートにドラッグ＆ドロップします。表示される設定画面で「OK」をクリック。ただし、その前に以下の設定を確認してください。Open an XAUUSD M5 chart. Drag and drop FusionGold M5 from the Navigator onto the chart. Click "OK" on the settings dialog. But first, check the settings below.

💡 ヒント 💡 Tip 「エキスパートアドバイザーの自動売買を許可」にチェックが入っていることを確認してください。Make sure "Allow Automated Trading" is checked in the Common tab.
04

Step 4: 入力パラメータを設定Step 4: Configure Input Parameters

EAのプロパティ画面で以下の主要パラメータを設定します：ロットサイズ（推奨: 0.01から開始）、リスク設定（デフォルト値を推奨）、各種フィルター設定。詳細は設定ガイドをご参照ください。In the EA properties, configure key parameters: Lot size (recommended: start at 0.01), Risk settings (defaults recommended), and various filter settings. Refer to the setup guide for details.

💡 ヒント 💡 Tip 最初はデフォルト設定のまま運用し、十分に動作を確認してから調整することをお勧めします。We recommend starting with default settings and adjusting only after confirming stable operation.
05

Step 5: 自動売買を有効化Step 5: Enable Auto Trading

チャート上部の「エキスパートアドバイザー」ボタン（スマイルアイコン）をクリックし、自動売買を有効にします。ボタンが緑色になれば、EAが稼働を開始します。Click the "Expert Advisor" button (smiley icon) on the chart toolbar and enable auto trading. When the button turns green, the EA is running.

💡 ヒント 💡 Tip EAが正しく動作しているかどうかは、チャート右上のスマイルマークで確認できます。Check the smiley face in the top-right corner of the chart to confirm the EA is working correctly.
06

Step 6: VPSでの運用（推奨）Step 6: Run on a VPS (Recommended)

24時間安定稼働させるにはVPSの使用を強く推奨します。主要なVPSプロバイダー（例：Forex VPS、AWS Lightsailなど）でWindows Serverを選択し、そのVPSにMT5とEAをインストールしてください。For 24/7 stable operation, we strongly recommend using a VPS. Choose a major VPS provider (e.g., Forex VPS, AWS Lightsail) with Windows Server, and install MT5 and the EA on it.

💡 ヒント 💡 Tip VPSの選択基準：IC Marketsへのレイテンシが低いリージョンを選びましょう（ロンドンまたはニューヨーク推奨）。VPS tip: Choose a region with low latency to IC Markets (London or New York recommended).
